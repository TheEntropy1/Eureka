# Eureka Web App
1. Install dependencies:
    pip install -r requirements.txt

2. Run the app:
    python app.py

3. Open your browser:
    http://127.0.0.1:5000/

Movie/Series/Anime Search + Stream free inside Eureka!
Default dark mode, Claude-AI inspired UI, shimmer loading.
